A plugin for IvсBox widgets.

== Description ==

IvcBox Widgets is a plugin that simplifies the installation of IvсBox widgets(https://ivcbox.com) on your website.

After activation of the plugin, you'll find *IvcBox Widgets* on the WordPress admin screen menu. With IvcBox Widgets, you are no longer need to worry about install IvcBox widget on your site. You will not need to ask your programmer about it.

For more detailed information, please refer to the https://ivcbox.com/contacts.

= Privacy Notices =

This plugin stores in the database a list of widgets that you can place on your site.

== Installation ==

1. Upload the entire `ivc-box-widget` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
